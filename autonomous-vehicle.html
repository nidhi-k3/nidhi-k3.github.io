Autonomous Navigation Robot

Indoor autonomous vehicle with LiDAR, RGB-D camera, and real-time obstacle detection

Role
Hardware Engineer Intern
Organization
Autonomous Vehicle Lab, NC A&T
Timeline
April 2025 - July 2025

Project Overview

I developed an embedded autonomous ground robot capable of indoor navigation through object detection and obstacle avoidance. The system integrates multiple sensors—2D LiDAR, RGB-D camera, and IMU—with an NVIDIA Jetson AGX Xavier for high-level perception and an Arduino Mega for low-level motor control.

Hardware Integration

  • Compute Platform: NVIDIA Jetson AGX Xavier running ROS2 (Robot Operating System)
  • Sensors: 2D RPLIDAR A3 (360° laser scanner), Intel RealSense D455 RGB-D camera, BNO055 IMU
  • Actuation: Arduino Mega 2560 interfaced with dual 550 RPM DC motors via dual H-bridge motor driver
  • Power System: Dual battery configuration—6-12V for motors, 12V/5V for Jetson/Arduino
  • Chassis: Differential drive with rear DC motors and front passive casters

Software & Perception Pipeline

I implemented a real-time sensor fusion pipeline that processes LiDAR point clouds and RGB camera data for obstacle avoidance and lane following with sub-100ms latency.

Computer Vision: I collected and manually annotated 3,200+ images to fine-tune the YOLOP (You Only Look Once for Panoptic Driving Perception) model for indoor navigation, achieving 92% accuracy in lane segmentation and drivable area detection.

Control Architecture: The Jetson runs perception algorithms and publishes velocity commands via ROS2 topics. The Arduino subscribes to /cmd_vel, converts commands into PWM signals (for speed) and digital direction signals, and drives the H-bridge motor controller.

Technical Challenges

  • Sensor Calibration: Performed intrinsic and extrinsic calibration for camera-LiDAR alignment to ensure accurate depth measurements
  • Real-Time Performance: Optimized perception pipeline to maintain <100ms latency while running YOLOv5 inference on Jetson GPU
  • Motor Control: Implemented differential drive kinematics, tuning PWM duty cycles to achieve smooth turns and straight-line motion
  • Data Pipeline: Built automated data collection and labeling workflow for training dataset

System Architecture

The robot uses a layered architecture:

  • Perception Layer (Jetson): Runs YOLOP for object detection, processes LiDAR scans, fuses sensor data
  • Decision Layer (Jetson): Implements obstacle avoidance logic and path planning
  • Control Layer (Arduino): Converts high-level velocity commands to motor PWM signals
  • Actuation Layer (H-Bridge): Amplifies control signals to drive DC motors

Key Learnings

  • Gained hands-on experience with ROS2, understanding publish-subscribe architecture and inter-node communication
  • Learned the importance of proper sensor mounting and vibration isolation for clean data
  • Developed skills in embedded systems debugging—oscilloscope usage, logic analyzer for I²C/PWM signals
  • Understood trade-offs between perception accuracy and real-time performance

Results

The final system successfully navigates indoor hallways, detects and avoids obstacles in real-time, and follows lane markings with 92% detection accuracy. The project demonstrated the feasibility of low-cost autonomous navigation using commodity hardware and open-source software.
